# Collectra

Collectra is a collector-first platform for trading card enthusiasts.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000 in your browser.

## Current milestone

This version contains the initial foundation only:
- Next.js with TypeScript
- Tailwind CSS
- App Router
- Premium dark landing page
- Basic project structure

## Project structure

- app/ - Next.js app routes and page components
- public/ - Static assets
- package.json - Project scripts and dependencies
